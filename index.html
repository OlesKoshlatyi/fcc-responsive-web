piano

In this project, I created a virtual piano using CSS and Responsive Design techniques to ensure it looks great on screens of all sizes. The webpage includes: FreeCodeCamp Logo: Displayed prominently at the top. Piano Keys: Implemented with div elements styled to resemble piano keys, including black keys for sharps. Responsive Design: Utilizes media queries to adjust the layout and styling based on different screen widths, ensuring a consistent user experience across devices. This project allowed me to apply CSS for layout and design, utilize media queries for responsiveness, and create an interactive element suitable for various screen sizes.

product lending

In this project, I created a product landing page that demonstrates my proficiency in HTML, CSS, and responsive design. The page includes: Header: A header element with an embedded logo image and a navigation bar. Navigation Bar: A nav element with clickable links that navigate to corresponding sections of the page. Product Video: An embedded video showcasing the product. Contact Form: A form with an email input field that includes HTML5 validation and a submit button, which sends the email to a mock URL. Responsive Design: The navbar stays fixed at the top of the viewport, and the layout adapts using CSS flexbox and media queries. This project highlights my ability to build structured, user-friendly, and responsive web pages with interactive elements and effective navigation.
